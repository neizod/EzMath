== วิธีติดตั้ง ==

สำหรับ Linux
วิธีที่ 1
ดาวน์โหลด math.l, math.y และ Makefile เก็บใน directory เดียวกัน
เข้าไปยัง directory ทาง terminal แล้วพิมพ์คำสั่ง make
*ต้องการโปรแกรม flex และ bison สำหรับคำสั่ง make

วิธีที่ 2
ดาวน์โหลด math.tab.h, math.tab.c และ lex.yy.c เก็บใน directory เดียวกัน
เข้าไปยัง directory ทาง terminal แล้วพิมพ์คำสั่ง
cc -o math math.tab.c lex.yy.c -lfl

== วิธีใช้ ==

เริ่มต้นใช้งานโปรแกรมด้วยคำสั่ง ./math ผ่าน terminal ใน directory ที่ทำการ make
พิมพ์นิพจน์ทางคณิตศาสตร์ที่ต้องการแปลลงไป กดปุ่ม enter/return อ่านผลลัพท์ที่บรรทัด (fin):::
ออกโปรแกรมโดยการกด Ctrl+D หรือ Ctrl+C

== รูปแบบไวยกรณ์ ==

case sensitive
เครื่องหมายวรรตอนใช้เพื่อพิจรณารูปแบบข้อความเท่านั้น ไม่ส่งผลกระทบต่อการแสดงผล

ตัวเลข
ใช้เลขอาราบิก 0-9
สำหรับเลขทศนิยม ต้องมีทั้งส่วนจำนวนเต็มและเศษส่วน .1 หรือ 4. จะถูกมองเป็นจุดและตัวเลขเท่านั้น

อักษรกรีก
สามารถพิมพ์ชื่ออักษรกรีกในภาษาอังกฤษลงไปได้เลย เช่น alpha Beta varphi

อักษรอังกฤษ
สามารถพิมพ์อักษรอังกฤษลงไปได้เลย เช่น a, B, xyz
ในกรณีที่ต้องการแสดงเป็นอักษรอังกฤษ แต่คำใช้นั้นซ้ำกับคำอื่น ให้ใช้การเว้นวรรคช่วยจัดกลุ่ม
เช่น z = y + xi ให้เขียนเป็น z = y + x i
