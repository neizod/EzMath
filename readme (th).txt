== วิธีติดตั้ง ==

สำหรับ Linux
วิธีที่ 1
ดาวน์โหลด math.l, math.y และ Makefile เก็บใน directory เดียวกัน
เข้าไปยัง directory ทาง terminal แล้วพิมพ์คำสั่ง make
*ต้องการโปรแกรม flex และ bison สำหรับคำสั่ง make

วิธีที่ 2
ดาวน์โหลด math.tab.h, math.tab.c และ lex.yy.c เก็บใน directory เดียวกัน
เข้าไปยัง directory ทาง terminal แล้วพิมพ์คำสั่ง
cc -o math math.tab.c lex.yy.c -lfl

== วิธีใช้ ==

เริ่มต้นใช้งานโปรแกรมด้วยคำสั่ง ./math ผ่าน terminal ใน directory ที่ทำการ make
พิมพ์นิพจน์ทางคณิตศาสตร์ที่ต้องการแปลลงไป กดปุ่ม enter/return อ่านผลลัพท์ที่บรรทัด (fin):::
ออกโปรแกรมโดยการกด Ctrl+D หรือ Ctrl+C

== รูปแบบไวยกรณ์ ==

case sensitive
เครื่องหมายวรรตอนใช้เพื่อพิจรณารูปแบบข้อความเท่านั้น ไม่ส่งผลกระทบต่อการแสดงผล

ตัวเลข
ใช้เลขอาราบิก 0-9
สำหรับเลขทศนิยม ต้องมีทั้งส่วนจำนวนเต็มและเศษส่วน .1 หรือ 4. จะถูกมองเป็นจุดและตัวเลขเท่านั้น

อักษรกรีก
สามารถพิมพ์ชื่ออักษรกรีกในภาษาอังกฤษลงไปได้เลย เช่น alpha Beta varphi
อักษรกรีกที่สามารถเขียนได้หลายแบบ สามารถเรียกได้โดยการเติม var นำหน้า

อักษรอังกฤษ
สามารถพิมพ์อักษรอังกฤษลงไปได้เลย เช่น a, B, xyz
ในกรณีที่ต้องการแสดงเป็นอักษรอังกฤษ แต่คำใช้นั้นซ้ำกับคำอื่น ให้ใช้การเว้นวรรคช่วยจัดกลุ่ม
เช่น z = y + xi ให้เขียนเป็น z = y + x i

เครื่องหมายดำเนินการทางคณิตศาสตร์
+ - * . ~ ! +- -+ (+) (-) (*) (.) (/)
= != < > << >> <= >= ~~ ~= == -> <-> and or not
sin cos tan cot sec csc asin acos atan acot asec acsc
Eset Nset Zset Qset Rset Cset aleph infinity ...
all exists in notin subset supset union intersect
เครื่องหมายและข้อความเหล่านี้จะถูกแปลเป็นรูปเครื่องหมายแบบต่างๆ

เครื่องหมายวงเล็บ
เครื่องหมายวงเล็บ ( ต้องการ ) เสมอ และไม่สามารถเขียน ) ก่อนหน้า ( ได้
เครื่องหมายวงเล็บในบางกรณีจะถูกใช้เพื่อจัดกลุ่มของนิพจน์ เช่น e^(pii) เมื่อแสดงผลเครื่องหมาย () จะหายไป
สำหรับกรณีที่เหลือ เครื่องหมาย () จะแสดงตามปรกติ เช่น (a+b)(x+y+z)

การหาร
รูปแบบ: นิพจน์1/นิพจน์2

การห้อยตัวชี้ตำแหน่ง
รูปแบบ: นิพจน์1[นิพจน์2]
*เครื่องหมาย [ ต้องอยู่ติดกับนิพจน์1 เท่านั้น มิฉะนั้นจะถูกแปลความเป็นเมทริกซ์

การหาผลรวม, อินทิกรัล
รูปแบบ: เครื่องหมายการหาผลรวม นิพจน์1
รูปแบบ: เครื่องหมายการหาผลรวม นิพจน์1 ประโยคประกาศขอบเขต
โดยที่

ประโยคประกาศขอบเขต: from นิพจน์1 to นิพจน์2

เมทริกซ์
รูปแบบ: [ นิพจน์11, นิพจน์12, ... , นิพจน์1n ; นิพจน์21, นิพจน์22, ... , นิพจน์nm ]
โดยที่
เครื่องหมาย , ใช้สำหรับแบ่งนิพจน์แต่ละนิพจน์ออกจากกัน
เครื่องหมาย ; ใช้สำหรับขึ้นต้นบรรทัดใหม่ในเมทริกซ์
